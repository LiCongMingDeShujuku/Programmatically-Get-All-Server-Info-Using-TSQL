## 中文
只需转到Management Studio，然后运行以下逻辑（logic）。
选择：
serverproperty(‘edition’)
, serverproperty(‘productlevel’)
, serverproperty(‘productversion’)
exec xp_msver;
exec xp_cmdshell ‘systeminfo’;

结果会提供很多有用的信息，如版本，服务器版本，安装日期，启动时间等。

## Logic
```SQL
select
serverproperty(‘edition’)
, serverproperty(‘productlevel’)
, serverproperty(‘productversion’)
exec xp_msver;
exec xp_cmdshell ‘systeminfo’;
```
The output will give you lots of good information such as the Edition, Server Version, Installation Date, Boot Time, etc.
